# Инструкция по созданию иконок для PWA

## Требуемые иконки:

1. **icon-192x192.png** - 192x192 пикселей
2. **icon-512x512.png** - 512x512 пикселей

## Как создать иконки:

### Вариант 1: Использовать онлайн генератор
1. Перейдите на https://www.pwabuilder.com/imageGenerator
2. Загрузите логотип NumerOM (минимум 512x512px)
3. Скачайте сгенерированные иконки
4. Поместите `icon-192x192.png` и `icon-512x512.png` в папку `/frontend/public/`

### Вариант 2: Создать в Figma/Photoshop
1. Создайте квадратное изображение 512x512px
2. Добавьте логотип NumerOM на фиолетовом фоне (#7c3aed)
3. Экспортируйте как PNG в двух размерах: 192x192 и 512x512
4. Сохраните файлы в `/frontend/public/` с названиями:
   - `icon-192x192.png`
   - `icon-512x512.png`

### Вариант 3: Использовать текущий favicon
Если у вас уже есть favicon или логотип:
1. Используйте https://realfavicongenerator.net/
2. Загрузите ваше изображение
3. Скачайте PWA иконки
4. Поместите в `/frontend/public/`

## Дополнительные иконки (опционально):

Для лучшей поддержки iOS:
- **apple-touch-icon.png** - 180x180px
- **favicon.ico** - 48x48px

## После создания иконок:

Иконки автоматически подключатся через `manifest.json`, который уже настроен.
